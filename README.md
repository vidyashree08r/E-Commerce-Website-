# E-Commerce-Website-
E-Commerce/Shopping Cart Web Application is a PHP/MySQL based online store source code that implements core e-commerce features such as product browsing by category, featured products, user registration &amp; login, shopping cart management, sorting ,pagination, and a checkout system.

# E-Commerce Website

## Project Description
This project is a PHP & MySQL based online shopping cart web application.

## Features
- User registration and login
- Product browsing by category
- Add to cart and checkout
- Admin panel for product management

## Technologies Used
- HTML, CSS, JavaScript
- PHP
- MySQL
- Apache (XAMPP)

## How to Run
1. Install XAMPP
2. Copy project to htdocs folder
3. Import database.sql in phpMyAdmin
4. Run http://localhost/project-name
